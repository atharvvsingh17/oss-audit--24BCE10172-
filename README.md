## Project Overview
[cite_start]This repository contains a comprehensive audit of the Python programming language, focusing on its open-source philosophy, licensing (PSF), and its functional footprint on a Linux (Ubuntu 24.04) system[cite: 10, 54].

## Shell Scripts Included
This audit features five custom Bash scripts covering Units 1–5 of the OSS course:
1. [cite_start]**script1.sh**: Reports system identity, kernel version, and OS distro using command substitution [cite: 134-136].
2. [cite_start]**script2.sh**: Inspects the installation status of Python3 using a `case` statement[cite: 139].
3. [cite_start]**script3.sh**: Audits file permissions for all project scripts using a `for` loop [cite: 207-208].
4. [cite_start]**script4.sh**: Analyzes root disk usage and provides health status via `if-else` conditionals [cite: 251-252].
5. [cite_start]**script5.sh**: An interactive confirmation tool using functions and user input to finalize the audit[cite: 274, 280, 288].
